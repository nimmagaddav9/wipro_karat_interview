JS
===========
/*
Given an API returning a list of todos, we want to fetch the list, create a separate block for each user, and display their todos in the appropriate block.

Use this endpoint URL to get the todos: https://dummyjson.com/todos?limit=10&skip=80. It will return the following structure with total of 10 todos:
{
  "todos": [
    {
      "id": 1,
      "todo": "Do something nice for someone I care about",
      "completed": true,
      "userId": 26
    },
  ],
}

Each block should contain the userId as the title of the block and the list of todos.
*/

fetch('https://dummyjson.com/todos?limit=10&skip=80')
 .then(res => res.json())
 .then(data => {
   const grouped = {};
   
   data.todos.forEach(todo => {
     if(!grouped[todo.userId]){
       grouped[todo.userId] = [];
     }
     
     grouped[todo.userId].push(todo);
   });
   
   const container = document.getElementById('users');
   
   Object.entries(grouped).forEach(([userId, todos]) => {
     const card = document.createElement('div');
     card.className = 'card';
     
    card.innerHTML = `
    <h3>User # ${userId}</h3>
    <ul>
    ${todos.map(todo => `<li>${todo.todo}</li>`).join('')}
    </ul>
    `;
    container.appendChild(card);
    });
 });

Angular Pipe Interview Questions & Answers

1.  How is the pipe made available to the module?

-   Declare the pipe in the module’s declarations array.
-   If using a SharedModule, export it and import that module where
    needed.

@NgModule({ declarations: [TruncatePipe] }) export class AppModule {}

2.  How do you use the pipe with default options? {{ content | truncate
    }}

Uses default values: - max = 10 - append = ‘…’

3.  How do you override the default parameters? {{ content |
    truncate:20:’***’ }}

-   20 becomes the max length.
-   *** becomes the appended text.

4.  How do you override only the append parameter? {{ content |
    truncate:10:’ [more]’ }}

-   Angular pipe arguments are positional.
-   You cannot skip the first parameter.
-   Pass the default max value and then the custom append value.

30-Second Interview Response: The pipe is made available by declaring it
in an Angular module. To use it with default settings, I’d write {{
content | truncate }}. To override parameters, I’d pass them using colon
syntax, such as {{ content | truncate:20:‘…’ }}. Since pipe arguments
are positional, if I only want to change the append text, I still need
to pass the max value first, for example {{ content | truncate:10:’
[more]’ }}.
